Date Methods

let currentDate =
new Date('March 15, 2009 21:05:34');

Date( )

currentDate.Date() -->
returns a string representation of the current date and time. All arguments are ignored.
The result is the same as executing new Date().toString().

getDate( )

currentDate.getDate() --> 15
returns the day as a numer (1 - 31).

getDay( )

currentDate.getDay() --> 3
returns the weekday as number (0 - 6).

getFullYear( )

currentDate.getFullYear() --> 2009
returns the year as a four-digit number (yyyy).

getHours( )

currentDate.getHours() --> 21
returns the hour (0 - 23).

getMilliseconds( )

currentDate.getMilliseconds() --> 0
returns the milliseconds (0 - 999).

getMinutes( )

currentDate.getMinutes() --> 5
returns the minute (0 - 59).

getMonth( )

currentDate.getMonth() --> 2
returns the month (0 - 11).

getSeconds( )

currentDate.getSeconds() --> 34
returns the seconds (0 - 59).

toString( )

currentDate.toString() -->
March 15, 2009 21:05:34 GMT-0400 (Eastern Daylight Time)
returns a string representing the specified Date object interpreted in the local timezone.



setDate( )

currentDate.setDate(25);
return currentDate.toString(); -->
Wed Mar 25 2009 21:05:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting date (1 - 31).

setYear( )

currentDate.setYear(2020, 10, 18)
return currentDate.toString(); -->
Wed Nov 18 2020 21:05:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting yyyy, mm, dd.
Month and date are optional.

setHours( )

currentDate.setHours(8)
return currentDate.toString(); -->
Wed Mar 15 2009 08:05:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting hours (0 - 23).

setMilliseconds( )

currentDate.setMilliseconds(135)
return currentDate.toString(); -->
Wed Mar 15 2009 21:05:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting milliseconds (0 - 999).

setMinutes( )

currentDate.setMinutes(42)
return currentDate.toString(); -->
Wed Mar 15 2009 21:42:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting minutes (0 - 59).

setMonth( )

currentDate.setMonth(8)
return currentDate.toString(); -->
Wed Aug 15 2009 21:05:34 GMT-0400 (Eastern Daylight Time)
returns date string after setting month (0 - 11).

setSeconds( )

currentDate.setSeconds(3)
return currentDate.toString(); -->
Wed Apr 15 2009 21:05:03 GMT-0400 (Eastern Daylight Time)
returns date string after setting seconds (0 - 59).
