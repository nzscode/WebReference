Math & Number Methods

let num1 = 12.3456;
let num2 = 98.7654
let num3 = [23.4567, 87.6543, 45.5445, 102.963852];


Number( )

"452" --> 452
returns a number string as a number

Number.parseFloat( )

Number.parseFloat("25.78 sheep") --> 25.78
returns a string with a number in front as the number only.
If not returns NaN.

Number.parseInt( )

Number.parseInt("25.78 sheep") --> 25
returns a string with a number in front as the number only, without any decimals.
If not returns NaN.

toString( )

num1.toString() --> 12.345
returns a number as a string.

Math.ceil( )

Math.ceil(num1) --> 13
returns a number that has been rounded up to the largest integer.

Math.floor( )

Math.floor(num1) --> 12
returns a number that has been rounded down to the smallest integer.

Math.max( )

Math.max (num3[0], num3[1], num3[2], num3[3])--> 102.963852

Using spread Operator:
Math.max(...num3) --> 102.963852

returns the maximum value among specified values.



Math.pow( )

Math.pow(2, 3) --> 8
returns the value of a base (2), raised to a power(3).

Math.sign( )

Math.sign(num1) --> 1
returns 1 or 0 if the specified value is a positive number.
returns -1 or -0 if the specified value is a negative number.

Math.random( )

Math.random() --> a random number between 0 & 1.
(changes on every refresh)

To get a random number between two specific numbers:
function getRandomNum(min, max) {
return Math.random() * (max - min) + min;
}
getRandomNum(1, 10) --> output between 1 inclusive and 10 exclusive (including decimal points).

To get a random integer between two specific numbers:
function getRandomNum(min, max) {
min = Math.ceil(min);
max = Math.floor(max);
return Math.floor(Math.random() * (max - min)) + min;
}
getRandomNum(1, 10) --> output between 1 inclusive and 10 exclusive.

returns a random number.

Math.round( )

Math.round(num1) --> 12
returns a number rounded to the nearest integer.

Math.trunc( )

Math.trunc(num1) --> 12
Math.trunc(num2) --> 98
returns the integer by removing any fractional digits.

Math.min( )

Math.min (num3[0], num3[1], num3[2], num3[3])--> 23.4567

Using spread Operator:
Math.min(...num3) --> 23.4567

returns the minimum value among specified values.
