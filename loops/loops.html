Loops

Loop Types:

For

Syntax

for (initialize; condition; update) {
statement/ return;
};

Example

for (i = 0; i < 5; i++) {
console.log(i);
};

0
1

while

Syntax

initialize;
while ( condition) {
statement/ return;
update;
};

Example

let i = 0;
while (i < 5) {
console.log(i);
i++;
};

Console Result

0
1
2
3
4
5

Do ... while

Executes at least once.

Syntax

initialize;
do {
statement/ return;
update;
}
while (condition);

Example

let i = 0;;
do {
console.log(i);
i++;
}
while (i < 5);

Console Result

0
1
2
3
4
5

Array Loops


Iterating through objects or arrays.

To iterate through the keys of an object:

For strings and arrays For...of is better.

Syntax

for (let key in object){
statement/return;
}

example

let student1 = {
name: "Henry",
age: 5
};

for (let key in student1){
console.log(key);
};

Console Result

name
age
